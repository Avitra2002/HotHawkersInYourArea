## Run the project
For this project, please use two terminals for frontend and backend

### Frontend
```
cd my-app
npm install
npm run dev
```

### Backend
```
cd backend-app
python -m .venv venv
source venv/bin/activate
pip install -r requirement.txt
python app.py
```

### Port
Frontend is accessible at http://10.32.4.205:3000 \
Frontend is exposed at port 5000
